# Ajou Online Judge

## Git Commit Message Convention
| Type     | Description              |
|----------|--------------------------|
| Feat     | 새로운 기능 추가                |
| Fix      | 버그 수정                    |
| Refactor | 코드의 로직 변경, 기능 개선         |
| Delete   | 코드나 파일 삭제                |
| Move     | 파일 이동, 파일명/디렉터리명 변경      |
| Test     | 테스트/벤치마크                 |
| Style    | 변수명/함수명 변경, 코드 포맷 변경     |
| Design   | UI 디자인 변경                |
| Deploy   | Docker, Nginx 등 배포 관련 작업 |
| Docs     | 문서 관련 작업                 |
| Chore    | 기타                       |
